Țiglă metalica - Banga (Găgăuzia si toată Moldova)

  • Țigla metalică “BANGA”este profilată din oțel de grosime 0.45 – 0.6 mm(ZN-225 g/m2)
  • Țigla metalică “BANGA” este un produs elegant si rezistent, îmbinând aspectul tradițional cu cel modern.
  • Țigla metalică “BANGA” este obținută din oțel zincat la cald și este protejată pe ambele părți de câteva straturi de protecție astfel încât nu apar fisuri în urma solicitărilor mecanice.
  • Țigla metalică “BANGA” este profilul classic de țiglă metalică cu cea mai mare popularitate si cerere. Este un profil modern, cu un aspect plăcut, foarte practic și usor de montat.
