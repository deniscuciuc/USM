Металлочерепица - Каскад (Гагаузия и вся Молдова)

  • Металлочерепица КАСКАД производится из стали толщиной 0,45 — 0,6мм (ЗН-225 г/м2).
  • Своим благородством металлочерепица КАСКАД привносит на рынок Республики Молдова характеристики классической черепицы комбинированной с устойчивостью к сезонным перепадам высоких и низких температур.
  • Металлочерепица КАСКАД — лучшее решение для крыш совмещая в себе эстетику с утилитарностью.
