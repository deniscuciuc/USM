Компоненты крыши (Аксессуары)

Комплект для крыши БРАВО ПРОФИЛЬ помимо покрытия и системы водоотвода включает аксессуары и предохранительные детали
которые обязательны для каждой крыши. Они необходимы для сборки крыши из черепицы, которая защитит ваше строение.
Аксессуары марки БРАВО ПРОФИЛЬ изготовлены из наиболее качественных материалов последнего поколения.

Конёк полукруглый большой

Покрывает верхушку крыши, предотвращая проникновение водных потоков внутрь.

Конёк полукруглый маленький

Альтернатива для большого конька, Покрывает верхушку крыши, предотвращая проникновение водных потоков внутрь.

Бордюр для стен

Используется у задних и разделительных стен. Предотвращает стекание воды по стенам.

Ендова

Устанавливается на стыке под покрытием для целенаправленной эвакуации воды.

Крепление листов к стенам

Устанавливается на пересечении водного потока с стеной для предотвращения проникновения воды на стену.

Крышка «грива»

Предназначена для заполнения пустот, а также для ограничения проникновения птиц.

Саморез кровельный

Закрепляют листы и аксессуары.

Битумная лента

Клейкая битумная самоклеющаяся лента для гидроизоляции.

Лента для заполнения пустот

Используется в eндова и крышке «гриве» для заполнения пустот.

Конусная заглушка

Используется для заполнения пустот и ограничения проникновения птиц.

Бордюр для фронтона

Предназначен для закрытия боковых сторон деревянного остова крыши. Монтируется поверх декоративной доски после закрепления черепицы.

Бордюр для фронтона под черепицу

Предназначен для закрытия боковых сторон. Монтируется поверх декоративной доски до закрепления черепицы.

Бордюр для карниза

Предназначен для связи с дренажной системой водостока для стекания воды с покрытия в жёлоб.

Заграждение для снега

Монтируется на покрытии и предотвращает движение снега по крыше.

Заграждение для снега Омега

Это альтернатива для условного заграждения для снега.

Разрыв для наклона

Предназначен для мест где воды меняют траекторию движения по крыше.

Комплект для дымохода

Монтируется у основания дымохода, тем самым предотвращая проникновение влаги в его основание и в стены.
